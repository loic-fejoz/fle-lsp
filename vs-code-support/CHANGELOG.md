# Changelog

All notable changes to the FLE LSP Client extension will be documented in this file.

## [0.0.1] - 2026-02-03

### Added
- Initial release of the FLE LSP Client.
- Support for **Fast Log Entry (FLE)** language.
- LSP integration providing:
    - **Diagnostics**: Real-time error and warning reporting for FLE syntax.
    - **Hover**: Context-aware information for QSOs.
    - **Completion**: Smart suggestions for bands, modes, and dates (including UTC time/date).
    - **Document Symbols**: Hierarchical view of the log (Year > Month > Day > QSO).
    - **Semantic Tokens**: Semantic syntax highlighting.
    - **Folding Ranges**: Fold logs by Year, Month, or Day.
    - **Formatting**: Automatic normalization of FLE lines.
- Custom "FLE Original" color theme.
